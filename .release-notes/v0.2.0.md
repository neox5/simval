# Release v0.2.0

Minor API release introducing an extension to the `Value` interface.

## Changes

**Value Package:**

- Added `WithTransforms` to the `Value` interface
  - Exposes the transform chain associated with a value
  - Enables inspection, tooling, and advanced composition

## Compatibility

- Minor API change due to interface extension
- Downstream implementations must add the new method

## Installation

```bash
go get github.com/neox5/simv@v0.2.0
```
